🚀 Quick Start Guide: Neural Core Terminal
Follow these steps to initialize your advanced command interface:

Download: Get the latest .exe build.

Workspace: Create a dedicated folder (e.g., Better Terminal) in your preferred directory.

Deployment: Move the .exe file into that folder.

Note: The application will automatically create history.txt and aliases.txt in this location.

Execution: Run the file. Your system is now ready.

🛠 Customization (Alias System)
Optimize your workflow by creating custom shortcuts directly in the terminal:

Command: add-alias name "command"

Example: add-alias myip "ipconfig | findstr /i ipv4"

Management: Use list-aliases to view all your saved shortcuts.
